FlexShop — фронтенд (Vue + Vite)

Запуск разработки:

```bash
cd frontend
npm install
npm run dev
```

Если нужно создать репозиторий на GitHub и запушить:

```bash
cd frontend
git init
git add .
git commit -m "Initial commit"
# если установлен GitHub CLI
gh repo create <owner>/<repo> --public --source=. --push
```
